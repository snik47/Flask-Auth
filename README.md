# Flask Authenticator (Python / Flask)

Simple authentication website with user registration and login.

## Run
In terminal copy and paste this:
- Windows: pip install -r requirements.txt
- macOS/ Linux: pip3 install -r requirements.txt

Run:
python app.py

Then open:
http://127.0.0.1:5000

## Features
- User registration
- Login and logout
- Password hashing
- Protected dashboard page
- SQLite database

## Notes
- Passwords are securely hashed
- Uses Flask-Login for session management
- Uses Flask-SQLAlchemy for database access
- Designed as a learning project for web authentication systems
